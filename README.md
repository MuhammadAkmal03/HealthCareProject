----------------------HealthCareAssistant-----------------------

The HealthCareAssistant project is a machine learning-powered system designed to assist in preliminary disease diagnosis. By analyzing key patient data like symptoms and vitals, it provides a data-driven prediction of a potential diagnosis.

Features 
Disease Diagnosis: Uses a trained machine learning model to predict common diseases.

Modular ML Pipeline: Built with a structured, reproducible workflow for data ingestion, transformation, training, and prediction.

Data-Driven Insights: Provides insights into model performance through a classification report and a confusion matrix.
